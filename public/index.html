How to use it

  1. Use Start studying to enter the compact reader-first layout.
  2. Reveal answer shows the back, and Hide answer conceals it again.
  3. Use ← / → to move, use the Difficulty menu or press 1–5 to rate the current card, and use Space / Enter to reveal or hide the answer, and use R to toggle reviewed status.
  4. Changing Difficulty saves immediately to cards.md on a 1–5 scale.
  5. Only the review controls update last_reviewed.
  6. Previous and Next only navigate the current session stack.

Start / stop

  • npm start runs the app normally.
  • npm run dev runs the app with watch mode.
  • Press Ctrl+C in the terminal to stop the app.
  • .bak/ stores startup backups and logs/ stores run logs.
