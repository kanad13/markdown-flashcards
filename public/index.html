How to use it

Keep the study flow simple and let the top banner carry the live context.

  • Use Start studying to enter the compact reader-first layout.
  • The top banner keeps four quick facts together: position, last reviewed, on-card time, and total session time.
  • Reveal answer shows the back, and Hide answer conceals it again.
  • Use ← / → to move, 1–5 to rate difficulty, Space / Enter to reveal or hide, and R to toggle reviewed status.
  • Difficulty and review changes save immediately to cards.md.

Configure cards.md

These settings shape the next session stack after the next server start. Keep them small and explicit.

  • filter_difficulty: [1, 2, 3] — include only the listed 1–5 difficulty values.
  • shuffle: yes or shuffle: no — choose shuffled order or file order.
  • exclude_reviewed_today: true — skip cards already reviewed when the session starts. It defaults to false.
  • Cards reviewed during the current session stay in the active stack even when exclude_reviewed_today is enabled.
